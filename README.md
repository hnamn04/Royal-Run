# Royal Run
3D Endless runner game made with Unity based on GameDevTV course.

## About The Project
Royal Run is a 3D endless runner game where the player avoids obstacles and enemies while surviving as long as possible.  
This project was created for learning Unity game development fundamentals such as movement systems, collision handling, prefabs, animations, and scene management.

## Features
- Character movement and controls
- Collectible item system (Coins and Apples)
- Score management system
- Procedural level generation
- Dynamic obstacle spawning
- Checkpoint system
- Collision detection and physics handling
- Endless runner gameplay
- Camera follow system

## Technologies Used
- Unity
- C#
- TextMesh Pro
- Git/GitHub

## What I learned
- Unity gameplay programming with C#
- Procedural map generation
- Object spawning systems
- Collision and physics handling
- Script organization in Unity
- Game manager and score systems
- Working with prefabs
